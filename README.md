# Elite Dental Care — Website

A premium dental clinic website built with React + Vite + Tailwind CSS v4.

## Features
- Animated hero slideshow
- Interactive before/after drag sliders
- Fully functional appointment booking modal
- Floating WhatsApp & call buttons
- Stats, services, team, and testimonials sections

## Run Locally

**Prerequisites:** Node.js 18+

1. Install dependencies:
   `npm install`

2. Run the dev server:
   `npm run dev`

3. Open: http://localhost:3000
